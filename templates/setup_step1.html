Step 1 — Telegram API

OpenMTOps reads trading signals from a Telegram channel. You need a free API key from Telegram.

How to get your API credentials:
1. Go to my.telegram.org/apps
2. Log in with your Telegram phone number
3. Create a new app (any name/platform)
4. Copy the App api_id (a number) and App api_hash (32 hex chars)
